### Packages

```bash
# Install
  brew install <package-name>

# Uninstall
  brew uninstall <package-name>
```

### Services

```bash
  # Start
  brew services start <service-name>

  # Restart
  brew services restart <service-name>

  # Stop
  brew services stop <service-name>
```
