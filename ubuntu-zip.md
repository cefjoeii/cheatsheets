---
title: Ubuntu Zip
---

### Gz - compress one file (only)

```bash
  # Compress file
  gzip <file-name>

  # Extract zip
  gunzip <zip-name>
```

### Tar - put multiple files into a single (tar) file

```bash
  # Compress files
  tar cvf <zip-name> <file-name-1> <file-name-2>

  # Extract zip
  tar xvf <zip-name>
```
