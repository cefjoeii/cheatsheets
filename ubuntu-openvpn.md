### VPNBook

```bash
# Download Free VPN config files
  Visit https://www.vpnbook.com/freevpn

# Extract and create three files inside the same directory
  ca.crt
  certificate.crt
  key.key

# Open one of the free VPN config files and copy each license enclosed by the appropriate tags to each of the three files
```

### OpenVPN

```bash
# Ensure that the following packages are installed
  network-manager-openvpn-gnome
  network-manager-openvpn
  network-manager-pptp
  network-manager-vpnc
```
